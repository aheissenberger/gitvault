## Requirement
- REQ-18

## Implementation
- **Unix (Linux/macOS):** `std::os::unix::fs::PermissionsExt::set_mode(0o600)` — sets owner read/write only.
- **Windows:** `icacls <path> /inheritance:r /grant:r <USERNAME>:F` — removes inheritance and grants full control to current user only.
- Both paths are in `src/permissions.rs::enforce_owner_rw`.

## Verification
- Unix permission restriction is validated in `src/permissions.rs` and `src/materialize.rs` tests.
- Windows ACL restriction is implemented in `src/permissions.rs` via `icacls` but has no CI coverage (platform limitation — Windows runner required).
