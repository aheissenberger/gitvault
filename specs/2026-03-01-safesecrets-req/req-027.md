## Requirement
- REQ-27

## Evidence
- `src/ssm.rs` `save_refs`: writes `HashMap<String, String>` of `key → /{app}/{env}/{key}` SSM paths to `secrets/{env}/.ssm-refs.json`. Values are never written.
- `src/ssm.rs` `cmd_ssm_pull`: populates refs from SSM parameter names only (not values).
- `src/ssm.rs` `cmd_ssm_set` / `cmd_ssm_push`: write values to AWS Parameter Store, then update local refs with only the path.
- Unit tests `test_serde_refs` and `test_load_refs_missing_file_returns_empty` verify the refs round-trip.
