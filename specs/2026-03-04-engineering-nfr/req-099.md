## Overview

`std::env::var("HOME")` fails on Windows where the user's home directory is stored in `USERPROFILE` (or obtained via `SHGetKnownFolderPath`). The `dirs` crate provides `dirs::home_dir()` which uses the platform-appropriate API on all three supported platforms.

## Change

Replace in `find_ssh_key_file`:

```rust
// Before — fails on Windows
let home = std::env::var("HOME").ok()?;
let ssh_dir = std::path::Path::new(&home).join(".ssh");

// After — cross-platform
let ssh_dir = dirs::home_dir()?.join(".ssh");
```

## Crate Justification

| Criterion | Result |
|-----------|--------|
| Correctness benefit | Fixes silent failure on Windows (HOME is absent) |
| Dependency weight | `dirs` adds 2 transitive dependencies |
| Ecosystem standing | Top-100 most downloaded crates on crates.io |
