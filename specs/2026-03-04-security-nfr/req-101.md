## Problem

`String::from_utf8_lossy` replaces invalid UTF-8 byte sequences with the Unicode replacement
character U+FFFD. This is appropriate for display purposes but not for security-sensitive comparisons:

1. **Leak detection false-negatives** ŌĆö `git diff --cached --name-only` and `git log --name-only`
   can return non-UTF-8 filenames on Linux (valid POSIX paths). If a sensitive path such as
   `.git/gitvault/plain/prod/secret.env` contains a non-UTF-8 byte in some component, lossy
   conversion replaces it, the string no longer contains `PLAIN_BASE_DIR`, and the leak check passes.

2. **Secret value corruption** ŌĆö `decrypt_env_secrets` converts the decrypted `Zeroizing<Vec<u8>>`
   to a `Cow<str>` via `from_utf8_lossy`. The `Zeroizing` wrapper drops, overwriting the byte buffer.
   The `Cow` (backed by `String`) escapes without zeroization and contains a corrupted value if any
   byte was non-UTF-8.

3. **SSH identity mismatch** ŌĆö `list_ssh_agent_keys` uses `ssh-add -l` output for fingerprint and
   comment matching. A non-UTF-8 comment causes the comment field to be corrupted; if the comment
   is used to locate an SSH key file, the wrong file (or none) is selected.

## Solution

Replace all `String::from_utf8_lossy` calls in security-sensitive paths with `String::from_utf8`
and propagate or skip on error:

- **Decrypted content (`paths.rs`)** ŌĆö return `GitvaultError::Usage` if the bytes are not valid
  UTF-8. Secret files must be UTF-8; non-UTF-8 content is an authoring error.
- **Git output for leak detection (`drift.rs`)** ŌĆö return `GitvaultError::Other`.
- **Git output for path resolution (`paths.rs`, `hooks.rs`)** ŌĆö return error or fall back gracefully.
- **SSH-agent output (`identity.rs`)** ŌĆö return `SshAgentError::NotAvailable`.
- **SSM remote URL (`ssm/refs.rs`)** ŌĆö fall back to directory-name.

## Implementation

- `src/repo/paths.rs:decrypt_env_secrets` ŌĆö `String::from_utf8(plaintext.to_vec())` wrapped in
  `Zeroizing<String>`.
- `src/repo/drift.rs:check_no_tracked_plaintext` ŌĆö `String::from_utf8(raw)?`.
- `src/repo/drift.rs:find_history_plaintext_leaks` ŌĆö `String::from_utf8(raw)?`.
- `src/repo/paths.rs:find_repo_root_from_with_runner` ŌĆö `String::from_utf8(out.stdout)?`.
- `src/repo/hooks.rs:resolve_hooks_dir` ŌĆö `String::from_utf8(out.stdout).map(|s| s.trim().to_string()).unwrap_or_default()`.
- `src/identity.rs:list_ssh_agent_keys` ŌĆö `String::from_utf8(raw).map_err(...)`.
- `src/ssm/refs.rs:get_app_name` ŌĆö `if let Ok(url_str) = String::from_utf8(raw)`.
