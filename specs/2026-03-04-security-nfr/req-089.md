## Overview
The HMAC token key material (32 bytes) is sensitive cryptographic material that must not linger in memory after use. This spec ensures that all intermediate representations of the key—both the binary array and its hex-encoded string form—are held in `Zeroizing` wrappers so their memory is securely overwritten when dropped.

## Key Changes

### load_or_create_token_key Function
All sensitive values in this function are wrapped in `Zeroizing`:

```rust
// Create or load the key in a Zeroizing wrapper
let key = Zeroizing::new(/* 32-byte array or decoded bytes */);

// For disk storage, encode in a Zeroizing String
let hex_string = Zeroizing::new(hex::encode(&*key));
std::fs::write(&key_path, hex_string.as_bytes())?;

// Return: copy the [u8; 32] out, then drop the Zeroizing wrapper
let result = *key;  // Copy occurs here
drop(key);  // Zeroized immediately after copy
Ok(result)
```

### Paths Affected

1. **Create path:** Generate a new random 32-byte key via `rand::thread_rng()`, store in `Zeroizing<[u8; 32]>`, encode as hex in `Zeroizing<String>`, write to disk, return the array.
2. **Load path:** Read file content in a `Zeroizing<String>`, hex-decode in a `Zeroizing<Vec<u8>>`, copy to a `Zeroizing<[u8; 32]>`, return the array.

## Why the Copy Is Unavoidable
The function returns `[u8; 32]` (a fixed-size array) because that is the input type for `hmac-sha256`'s `HmacSha256::new()` constructor. Copying the bytes from `Zeroizing` to the return value is necessary; the Zeroizing wrapper is dropped immediately after, zeroing its memory. The short-lived stack copy is acceptable because it is out of scope within microseconds.

## Testing
- **Unit tests:** Verify that key load/creation completes and HMAC verification succeeds.
- **Memory inspection (optional):** Use tools like `valgrind` with `--leak-check=full` to confirm zeroization behavior (advanced).

## Risk & Mitigation
- **Return type unchanged:** `[u8; 32]` remains the same; no caller updates needed.
- **Behavioral impact:** None; security is improved with zero behavioral changes.
